Iraq E-Visa Form Filler

Enter the URL of your Passport Control Admin application (without trailing slash)

📋 Instructions

  1. Enter the URL of your deployed Passport Control Admin application
  2. Click "Test Connection" to verify the API is accessible
  3. Save your settings
  4. Navigate to eservice.evisa.iq
  5. Click the extension icon and select a passenger to auto-fill the form

🔒 Privacy

This extension only communicates with your configured Passport Control Admin server. No data is sent to any third parties.
